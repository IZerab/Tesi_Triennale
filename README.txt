This is the base for the Bachelor Degree thesis of Lorenzo Beltrame for his Physics degree at the University of Trento.
The thesis supervisor is prof. Luca Tubiana.
The tile is "A machine learning approach to forecast Ozone concentration in Trentino".
The thesis focuses on enviromental and athmospheric data to predict hazardous pollutants' concentration in several towns.
The project is based upon data supplied by Trentino's enviromental Agency (APPA).
The data can be found here: https://bollettino.appa.tn.it/aria/

The project is written in python (v. 3.9).

There is the main file and several python file that serve as helper function.
